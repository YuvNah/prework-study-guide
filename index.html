CSS

  • CSS holds the design information of the webpage.
  • more CSS note here...
  • The Selector defines the element to which the rules or declarations apply (ex: div, span...).
  • Declarations have two important components: the property (color) and the value (Blue).
  • The (*) selector applies to everything that is visible on the page
  • Connect the CSS sheet in the head using: link rel="stylesheet" href="./assets/style.css" (make sure to save the css file in the right file so that you can access it)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
